# Phrases Web Client (CRUD)

Frontend React para consumir sua API .NET com operações CRUD.

## Requisitos
- Node 18+
- Backend (PhrasesApi) rodando em http://localhost:5000

## Instalação
1. npm install
2. Crie um arquivo .env na raiz com: VITE_API_URL=http://localhost:5000/api/phrases
3. npm run dev

## Funcionalidades
- Listar frases
- Criar nova frase
- Editar frase
- Deletar frase
- Animação: frases voando na tela
